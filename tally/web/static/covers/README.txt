Drop a cover image here named after the hobby key to replace its generated
placeholder, e.g.:

    quartiles.png   crossword.png   set.png   anagrams.png
    word_hunt.png   dialed_sound.png   paper.png

png, jpg, jpeg, webp, and svg all work; the file is used as-is on the gallery.
Until a file exists, tally draws a placeholder cover (see tally/web/covers.py).
Hobby keys live in tally/hobbies.py.
