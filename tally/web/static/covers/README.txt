Cover art for the gallery, keyed by hobby name (keys live in tally/hobbies.py).

Drop a file here named after the hobby to replace its generated placeholder:

    covers/<hobby>.png
    e.g.  covers/quartiles.png   covers/movie.png

Resolution for any cover: covers/<hobby> if present, else a generated
placeholder (tally/web/covers.py) drawn in the neutral accent (games) or the
hobby's own colour (everything else).

png, jpg, jpeg, webp, and svg all work. Posters look best at a 2:3 ratio.
